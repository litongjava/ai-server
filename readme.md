# ai-server

## 简介

ai-server 一款基于Java的提供高性能AI服务器的服务端

ai-server A Java-based server that provides a high-performance AI server

- Java
- 机器推理框架 djl
- Web中间件 tio



# 安装
启动

使用mvn和spring-boot插件启动
```
mvn clean package -DskipTests spring-boot:run
```
